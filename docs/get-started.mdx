### Download Clients

### Install

Install the desktop server and run

<img src="/img/run-server.png" alt="run server" width="400" />

### Run the Mobile App

Open the mobile app and tap on the Plus Icon. **Make sure your mobile and desktop are connected to the same wifi network**

<img src="/img/home.jpg" alt="Home Screen" width="300" />

### Scan QRCode

Scan the QRCode displayed on the desktop client

<img src="/img/scan-qr.jpg" alt="QRCode scan" width="300" />

### Connect To Server

Tap on the server and press connect

<img src="/img/connect-server.jpg" alt="Connect Server" width="300" />

### Take control

Control your mouse and keyboard from mobile!

<img src="/img/touchpad.jpg" alt="touchpad" width="300" />

If you face any issues connecting take a look at the [Troubleshooting](/docs/troubleshooting) section
